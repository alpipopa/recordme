# 📇 نظام تسجيل بيانات الأشخاص (Contact Management System)

نظام إداري متكامل مصمم بلغة PHP وقواعد بيانات MySQL لإدارة سجلات الأشخاص، الأرشفة الإلكترونية، وإصدار التقارير الرسمية.

## 🚀 المميزات الرئيسية

*   **👥 إدارة الأشخاص**: إضافة، تعديل، وعرض بيانات الأشخاص بتصنيفات مخصصة.
*   **📂 الأرشفة الإلكترونية**: إمكانية رفع صور الهويات وعدد غير محدود من المستندات (PDF, Word, صور) لكل شخص.
*   **🔧 حقول مخصصة**: إنشاء حقول إضافية ديناميكية (نصوص، قوائم، مربعات اختيار) لتناسب احتياجاتك البرمجية دون تعديل الكود.
*   **🖨 نظام طباعة احترافي**:
    *   دعم الطباعة الرأسية (Portrait) والأفقية (Landscape).
    *   إمكانية رفع شعارات ورأس وتذييل (Headers & Footers) كصور منفصلة لكل اتجاه طباعة.
    *   تخصيص الحقول المراد طباعتها في التقارير.
*   **👤 إدارة المستخدمين**: نظام صلاحيات (مدير، مدير مستخدمين، مشاهد) مع إمكانية رفع صور شخصية للمستخدمين.
*   **🔔 نظام تنبيهات**: تنبيهات فورية داخلية عند إضافة أشخاص جدد أو إجراء عمليات هامة.
*   **🛡 أمان عالي**: حماية ضد ثغرات CSRF، SQL Injection، ونظام جلسات آمن.

---

## 🛠 متطلبات التشغيل

*   **خادم ويب**: Apache أو Nginx (ينصح بـ Laragon أو XAMPP).
*   **لغة PHP**: إصدار 8.0 أو أعلى.
*   **قواعد البيانات**: MySQL/MariaDB 5.7 أو أعلى.
*   **إضافات PHP المطلوبة**: `PDO`, `GD`, `Fileinfo`.

---

## 📥 طريقة التثبيت

1.  **تحميل الملفات**: قم بفك ضغط ملفات النظام داخل مجلد الـ `www` أو `htdocs`.
2.  **إعداد قاعدة البيانات**:
    *   أنشئ قاعدة بيانات جديدة باسم `contact`.
    *   استورد الملف الموجود في `sql/Contact.sql` .
3.  **ضبط الإعدادات**:
    *   افتتح ملف `config/db.php`.
    *   قم بتعديل بيانات الاتصال (المضيف، اسم المستخدم، كلمة المرور).
    *   قم بتحديث `APP_URL` ليتناسب مع رابط تشغيل النظام لديك (مثال: `http://localhost/contact`).
4.  **صلاحيات الملفات**:
    *   تأكد من أن مجلد `uploads` والمجلدات الفرعية داخله قابلة للكتابة.

---

## 🔐 بيانات الدخول الافتراضية

*   **رابط لوحة التحكم**: `your-domain.com/admin/dashboard.php`
*   **اسم المستخدم**: `admin`
*   **كلمة المرور الافتراضية**: `Admin@2024`
*   *(يرجى تغيير كلمة المرور فور الدخول من صفحة الملف الشخصي)*.

---

## 📁 هيكلية المجلدات

*   `admin/`: تحتوي على صفحات لوحة التحكم (إضافة أشخاص، تقارير، إعدادات).
*   `includes/`: تحتوي على ملفات الدوال البرمجية، الأمان، والهيدر والفوتر.
*   `config/`: ملفات إعدادات قاعدة البيانات والثوابت.
*   `assets/`: ملفات التصميم (CSS) والبرمجة (JS).
*   `uploads/`: مكان تخزين صور الهويات والمستندات المرفوعة.
*   `sql/`: ملفات قواعد البيانات والهجرات.

---

## 📝 ملاحظات للمطورين

*   النظام يستخدم Vanilla CSS و Vanilla JS لضمان الخفة والسرعة القصوى.
*   تم اتباع أسلوب البرمجة الإجرائية المنظمة لتسهيل الفهم والتعديل.
*   جميع العمليات الهامة يتم تسجيلها في `audit_log` لمراقبة نشاط المستخدمين.
